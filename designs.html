Supplementary Material for Grid-block Difference Families (DFs)

Supplementary data for my paper "Grid-block difference families and related combinatorial structures" (co-author with J. Satoh and M. Jimbo).

$q=p<105,841$ prime

  • There are $837$ primes satisfying the necessary divisibility condition $p \equiv 1 \pmod{36}$ and above condition.
    • Here is the data file (.txt, 20KB). Each row gives a row-radical $(p, L_{3,3},1)$-DF.
    • The parameters in each row (separated by commas) are respectively $p$,   $n=\frac{p-1}{36}$,   $g$,   $\zeta_3$,   $\alpha_2$,   $\alpha_3$.
    • $p$: order the group,   $g$: a primitive element of $\mathbb{F}_p$,   $\zeta_3$: a cubic root of unity of $\mathbb{F}_p^{\ast}$.
    • $\alpha_2$, $\alpha_3$: coefficients in the 2nd and 3rd row of a base grid-block. (We always set $\alpha_1=1$.)

$q = p^3$, $q = p^4$ or $q = p^6$ with $p$ prime

  • $p = 13$,   $q = p^3 = 2197$.   $\mathbb{F}_{p^3} := \mathbb{F}_{p}[\beta] / (\beta^3 + \beta + 6)$.
    • $\zeta_3 = 9$,   $\alpha_2 = \beta^2$,   $\alpha_3 = \beta^2 + 5 \beta + 1$.  (We always set $\alpha_1=1$.)
  • $p = 17$,   $q = p^4 = 83521$.   $\mathbb{F}_{p^4} := \mathbb{F}_{p}[\beta] / (\beta^4 + 7\beta^2 + 10\beta + 3)$.
    • $\zeta_3 = 6\beta^3 + 12\beta^2 + 5\beta + 10$,   $\alpha_2 = \beta+1$,   $\alpha_3 = 2 \beta + 7$.  (We always set $\alpha_1=1$.)
  • $p = 5$,   $q = p^6 = 15625$.   $\mathbb{F}_{p^6} := \mathbb{F}_{p}[\beta] / (\beta^6 + \beta + 2)$.
    • $\zeta_3 = 3 \beta^5 + \beta^4 + 2 \beta^2 + 2\beta + 2$,   $\alpha_2 = \beta^2$,   $\alpha_3 = \beta^3 + 4 \beta^2 + 4 \beta $.  (We always set $\alpha_1=1$.)
